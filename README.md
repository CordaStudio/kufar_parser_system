# Telegram-бот для автоматического сбора объявлений kufar.by


## О проекте
Данный проект представляет собой Telegram-бота, предназначенного для автоматического мониторинга и сбора объявлений, размещаемых на сайте kufar.by.

Бот регулярно проверяет страницы сайта, находит новые объявления и отправляет их пользователю в Telegram. Это позволяет оперативно получать информацию о появлении новых предложений без необходимости вручную отслеживать обновления на сайте.

Проект был разработан как инструмент автоматизации для быстрого и удобного получения новых объявлений.


## Задача
Перед разработкой стояли следующие требования:
1. Обеспечить максимально быстрое получение новых объявлений;
2. Минимизировать системные требования для возможности запуска на недорогом сервере;
3. Ограничить доступ к чат-боту для предотвращения использования посторонними пользователями.


## Используемые технологии
• Python - основной язык разработки;

• Aiogram - асинхронный фреймворк для разработки Telegram-ботов;

• AioHTTP - библиотека для выполнения асинхронных HTTP-запросов;

• BeautifulSoup4 - парсинг HTML и извлечение данных со страниц;

• SQLite - лёгкая база данных для хранения информации об объявлениях.


## Основной функционал
Бот реализует следующий набор возможностей:
1. Автоматический мониторинг страниц сайта kufar.by;
2. Поиск и обработка новых объявлений;
3. Отправка найденных объявлений пользователю в Telegram;
4. Хранение уже обработанных объявлений для предотвращения повторной отправки;
5. Ограничение доступа к боту для авторизованных пользователей.


## Архитектура решения
Работа бота построена на асинхронной архитектуре.
1. Планировщик запускает периодическую проверку страниц сайта;
2. AioHTTP выполняет асинхронные HTTP-запросы;
3. BeautifulSoup4 извлекает данные объявлений из HTML;
4. Полученные объявления сравниваются с данными в базе SQLite;
5. Если объявление новое - бот отправляет его пользователю в Telegram.

Такая архитектура позволяет эффективно использовать ресурсы сервера и быстро обрабатывать новые объявления.

## Результат
В результате разработки был создан Telegram-бот, который полностью автоматизирует процесс мониторинга объявлений на kufar.by.
Решение обеспечивает:
1. Быстрое получение новых объявлений;
2. Низкую нагрузку на сервер;
3. Простоту развертывания и эксплуатации.
Бот успешно выполняет поставленные задачи и позволяет пользователю оперативно получать новые объявления без ручного мониторинга сайта.
